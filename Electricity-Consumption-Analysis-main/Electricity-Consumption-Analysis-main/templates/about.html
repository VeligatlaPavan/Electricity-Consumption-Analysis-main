SQLite Table Schema

The SQLite schema maps exactly to the fields in the historical electricity consumption CSV dataset.

Column Name SQL Type Description
States TEXT Indian State or Union Territory
Regions TEXT Regional power grid code (e.g. NR, WR, SR)
Latitude REAL State center latitude coordinate
Longitude REAL State center longitude coordinate
Dates DATE Observation date (format: YYYY-MM-DD)
Usage REAL Daily power consumption in Mega Units (MU)

Performance & Analytics Hub

Metrics detailing system calculations and visualization dimensions required for evaluation.

  • Amount of Data Rendered to DB: 16,599 total rows ingested cleanly. Indeces applied for date-based ranges.
  • Utilization of Data Filters: Region classifications (5 distinct values), calendar years, and date range indices.
  • No of Calculation Fields: Calculated average monthly consumption, growth recovery ratios, and regional totals.
  • No of Visualizations/Graphs: 3 native interactive Chart.js widgets, 1 embedded Tableau Dashboard, and 1 Tableau Story.
