An Exploration of Electricity Consumption Patterns

An in-depth data analytics study analyzing state-wise electricity consumption trends across India. Spanning from January 2019 to December 2020, this analysis captures seasonal variations, regional demands, and the historic impact of the nationwide COVID-19 lockdown on industrial and residential energy usage.

Core Analysis Scenarios

Scenario 1: Overall Trends

Track month-by-month national consumption patterns. Analyze demand spikes during peak summer months and the significant contraction in power usage during the nationwide lockdown (March-June 2020).

Scenario 2: Regional Demands

Compare uneven demand distributions between Northern, Southern, Eastern, Western, and Northeastern grid regions. Explore how varying climate profiles and industrial density drive electricity needs.

Scenario 3: Lockdown Recovery

Investigate how quickly economic activities and power demands rebounded across states. Compare post-lockdown averages to pre-lockdown levels to understand regional recovery paces.

Project System Architecture

Advanced Analytics Pipeline

This platform showcases how raw energy time-series datasets can be parsed, loaded into a relational SQL database, queried for critical metrics, and beautifully visualized using dynamic dashboards.

  • Pandas & SQLite clean, format, and store 16,599 records cleanly.
  • NumPy Matrix Math calculates post-lockdown rebound growth metrics.
  • Matplotlib & Plotly power static reports and high-speed web charts.
  • Flask Web Server provides dynamic database querying and API endpoints.
