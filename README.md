# ⚡ Electricity Consumption Analysis

## 📖 Project Overview

Electricity plays a vital role in powering industries, businesses, and households. Every day, large volumes of electricity consumption data are generated across different states and regions. Extracting meaningful insights from this data is essential for understanding usage patterns, improving energy efficiency, and supporting data-driven decision-making.

This project analyzes electricity consumption data using **Oracle SQL** for data management, **Tableau** for interactive visualizations, and **Flask** for developing a user-friendly web application. The system transforms raw electricity data into interactive dashboards that help users explore trends, compare regional consumption, and identify key insights.

## 🚀 Key Features

* 📊 Interactive dashboards for electricity consumption analysis
* 🗄️ Efficient data storage and querying using Oracle SQL
* 📈 Trend analysis across different states and regions
* 🔍 Data filtering and visualization with Tableau
* 🌐 Flask-based web application for easy access to insights
* 📉 Performance analysis and data-driven reporting

## 🛠️ Technologies Used

* **Oracle SQL** – Database Management
* **Tableau** – Data Visualization
* **Flask** – Web Application Framework
* **Python** – Backend Development

## 🎯 Project Objective

The primary objective of this project is to simplify electricity consumption analysis by converting raw data into meaningful visual insights. The application enables users to monitor consumption trends, compare regional electricity usage, and support better energy management through interactive dashboards.

## 📌 Outcome

This project demonstrates how database management, data visualization, and web technologies can be integrated to build an efficient electricity consumption analysis system that is informative, scalable, and easy to use.
